ระบบร้องเรียนออนไลน์ Smart Complaint

ระบบอัจฉริยะที่ช่วยรับเรื่องราวร้องทุกข์จากประชาชน ตอบสนองรวดเร็ว สามารถติดตามงานได้ และแบ่งหน้าต่างการใช้งานสำหรับเจ้าหน้าที่อย่างเป็นระบบ

Landing Page

หน้าหลักสำหรับประชาชน

ออกแบบหน้าต้อนรับให้สะอาดตาและใช้งานง่าย นำเสนอเนื้อหาครบถ้วน ประชาชนสามารถกดเข้าไปเพื่อเริ่มกรอกฟอร์มหรือติดตามสถานะของตัวเองได้ทันทีในคลิกเดียว

User Form

แบบฟอร์มแจ้งร้องเรียน

แบบฟอร์มออนไลน์ที่ช่วยให้ประชาชนกรอกข้อมูลได้สะดวก แบ่งสัดส่วนข้อมูลส่วนตัวและรายละเอียดเรื่องร้องเรียนอย่างชัดเจน มีระบบตรวจสอบความถูกต้องเพื่อลดข้อผิดพลาด

Admin Dashboard

หน้าต่างผู้ดูแลระบบ

แผงควบคุม (Dashboard) สำหรับเจ้าหน้าที่ ใช้ประมวลผลสถิติ อัปเดตสถานะของเคสต่างๆ และจัดการเรื่องร้องเรียนที่เข้ามาแต่ละวันได้อย่างรวดเร็วและมีประสิทธิภาพตรวจสอบได้

ระบบพิกัดแจ้งเหตุ Chaiyaphum Smart EMS

โปรเจกต์พัฒนาระบบบริหารจัดการด้านการแพทย์ฉุกเฉินอัจฉริยะ (Smart EMS) โดยผสานการทำงานของระบบฐานข้อมูลผู้ป่วย เข้ากับระบบแผนที่อัจฉริยะ เพื่ออำนวยความสะดวกในการเข้าช่วยเหลืออย่างทันท่วงที

Location API

ค้นหาและระบุตำแหน่งผู้ป่วย

อินเทอร์เฟซอัจฉริยะแบ่งหน้าจอเป็นส่วนแผนที่และการค้นหา ช่วยให้เจ้าหน้าที่ค้นหาข้อมูลหรือรหัสผู้ป่วยได้อย่างรวดเร็ว ระบบจะแสดงหมุดพิกัดบนแผนที่พร้อมหน้าต่างสรุปข้อมูลสมาชิกในบ้าน และปุ่มนำทางฉุกเฉิน (Google Maps) ทันที

Patient Directory

รายการข้อมูลฉุกเฉินผู้ป่วย

ฟีเจอร์แสดงผลการค้นหาแบบรายการ พร้อมป้ายกำกับ (Badges) แจ้งเตือนสถานะผู้ป่วยติดเตียง และโรคประจำตัวด้วยสีสันเร่งด่วนที่ชัดเจน ช่วยให้รถพยาบาลและทีมแพทย์เตรียมการปฐมพยาบาลได้อย่างแม่นยำก่อนลงพื้นที่จริง
